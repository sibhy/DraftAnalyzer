Project Description:
This project lets users simulate fantasy football drafts using AI with different strategies. 
Users can also customize the number of teams and which pick in the draft they have. 
Along with that, users can separately look up players they are interested in to see their stats and more. 

How to Run: 
1. Right click the zip file and select extract all
2. Run 'runthisfile.py' 

Libraries: 
This project uses pandas, beautifulsoup, numpy, and urllib3. 

Shortcut Commands: 
Pressing 0 in the splash page will bring you directly to the player lookup screen. 
Pressing - in the splash page will bring you directly to the draft selection screen.
